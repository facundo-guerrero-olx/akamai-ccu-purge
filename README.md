akamai-ccu-purge
================

Just a simple script to flush Akamai Cache


# Flush a URL
	python purge.py -U username@foobar.org -P superSecurePassword -u http://www.foobar.org

# Flush a CPCODE

	python purge.py -U username@foobar.org -P superSecurePassword -c CPCODE
